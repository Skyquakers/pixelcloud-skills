# pixelcloud-skills

A collection of AI agent skills for [PixelCloud](https://pixelcloud.cn) — making game server management accessible to AI coding assistants.

## What's Inside

| Skill | Description |
|---|---|
| [pixelcloud-cli](skills/pixelcloud-cli/SKILL.md) | Gives your AI agent the ability to manage multiplayer game servers via the `pixel` CLI — create, start, stop, restart, delete servers, search images, and more. |

## Usage

```bash
npx skills add skyquakers/pixelcloud-skills
```
